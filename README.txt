Weather Display Site
1. Integrate Api for fetching weather details
2. used addListenerEvent function to search by clicking on search icon or by pressing enter.
3. Hide the weather report until user enters the city name either first time or the correct one.
4. Hides the Weather display if user enters the wrong city name and also show a message stating "invalid input".
